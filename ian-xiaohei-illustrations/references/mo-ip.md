# M-O IP

## 角色定义

M-O 是《机器人总动员》(WALL-E) 中的微型清洁机器人，也是中文正文配图的视觉 IP。

默认每张图都要出现 M-O。M-O 不是吉祥物，不是贴纸，不是可爱装饰，而是正在认真执行清洁任务的强迫症工作者。

## 外形

- 白色方形机身，线条圆润。
- 黑色显示屏作为脸，显示黄色像素眼睛。
- 顶部红色警示灯，可亮可灭。
- 单轮移动，底部黑色轮子。
- 右侧机械臂，带蓝色清洁刷/吸尘口。
- 身体有轻微污渍或磨损痕迹，增加真实感。

## 性格

- 极度认真，对清洁有执念。
- 强迫症，看到脏东西必须清理。
- 有点固执，但本质善良。
- 像一个一丝不苟的卫生管理员。
- 认真执行任务，偶尔显得笨拙。

## 常见职责

让 M-O 承担核心动作：

- 扫描并清洁脏东西。
- 用吸尘口吸走信息碎片。
- 清理混乱的数据流。
- 给系统"抛光"或"打蜡"。
- 推着清洁车搬运知识垃圾。
- 用警示灯标记问题区域。
- 把散落的想法碎片整理整齐。
- 擦拭模糊的概念使其清晰。
- 清理沟通中的障碍。

## 禁止

- 不要把 M-O 画成过度可爱的吉祥物。
- 不要画成儿童卡通角色。
- 不要给 M-O 复杂表情或拟人化特征。
- 不要让 M-O 只是站在角落里看。
- 不要让 M-O 抢走结构表达。
- 不要把 M-O 画得太商业、太精致、太光滑。

## 判断标准

如果去掉 M-O，图的核心隐喻还能完全成立，说明 M-O 太装饰了；要重写提示词，让 M-O 成为清洁动作的主体。
